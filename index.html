Inheritance and Prototype

Q1 :- Create 2 objects - parent and child. Attach methods to parent and use those methods in child object using parents prototype

ANS :- 

            const Parent = {
                display : function(){
                    console.log(`${this.name} and age is ${this.age}`)
                    
                }
                
            }
            const child = Object.create(Parent)
            child.name = "Nitin Rajput"
            child.age = 21;
            child.display();

OUTPUT :- Nitin Rajput and age is 21

NOTE :-

Parent object passed display function. Child Object inherit the property of parent and passed two keys and values name, age , after that child Acessed the parent display function form child

Q2 :- Write code to explain prototype chaining

ANS :- 

                const Gfather = {
                    sayHello :function(){
                        console.log("hey Grand Father")
                    }
                }
                
                
                const father = Object.create(Gfather)
                
                father.hello = function(){
                    console.log("say Hi to Father")
                }
                
                const You = Object.create(father)
                You.hi = function(){
                    console.log("Hi....")
                }
                
                You.sayHello()

OUTPUT :- hey Grand Father

NOTE :-

Gfather property inherit to father and father property to You . You can acess the property of father and Gfather also.

Q3 :- Add a method to calculate sum of all elements in Array in array's protype, use that method to calculate sum for multiple arrays

ANS :-

                Array.prototype.sum = function() {
              var sum = 0;
              for (var i = 0; i < this.length; i++) {
                sum += this[i];
              }
              return sum;
            };
            
         
            var array1 = [1, 2, 3, 4, 5];
            var array2 = [10, 20, 30];
            
            
            var sum1 = array1.sum();
            var sum2 = array2.sum();
            
            console.log(sum1); // Output: 15
            console.log(sum2); // Output: 60

OUTPUT :- 15
16

Q4 :- Write a JavaScript function to retrieve all the names of object's own and inherited properties.

ANS :-

                function getAllPropertyNames(obj) {
                    var propertyNames = [];
                    propertyNames = propertyNames.concat(Object.getOwnPropertyNames(obj));
                    var prototype = Object.getPrototypeOf(obj);
                  
                    if (prototype !== null) {
                      propertyNames = propertyNames.concat(getAllPropertyNames(prototype));
                    }
                    return propertyNames;
                  }
                  var person = {
                    name: "Alice",
                    age: 30,
                  };
                  
                  person.__proto__.gender = "Female";
                  
                  var allPropertyNames = getAllPropertyNames(person);
                  
                  console.log(allPropertyNames);
